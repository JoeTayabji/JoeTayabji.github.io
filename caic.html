CAIC Accident Explorer

This project was a collaboration between Colorado Mountain College (CMC) and the Colorado Avalanche Information Center (CAIC). This partnership aimed to create a new Avalanche Accident Explorer, which displays spatial information about fatal avalanches in Colorado. I worked with Dr. Dara Seidl and Lucy Kepner on this project. This interactive tool can be found here, and the press release here.

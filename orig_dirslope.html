Directional Slope: A Python Tool For Path Design

This script is designed to calculate the directional slope of a line. It takes two inputs: a line and a DEM raster, both of which must be projected. The output is a new line with a directional slope attribute field. The user can specify a sampling distance. The input line will be split at this distance into new segments. The directional slope is calculated at the midpoint of the new segment. This script uses Horn's method for calculating slope at any location (Horn, 1981). It then uses a basic trigonometric calculation to adjust for directionality (Neteler and Mitášová, 2008).

This project, a collaboration with Dr. Dara Seidl, is now being developed into separate QGIS and ArcGIS plugins.
